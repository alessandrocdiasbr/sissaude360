# SisSaúde360 - Frontend

Este é o módulo frontend da plataforma SisSaúde360, desenvolvido em **React** com **TypeScript** e **Vite**.

## 🛠️ Como Iniciar

Para as instruções completas de como iniciar o sistema (Frontend + Backend + Banco de Dados) utilizando **Docker**, consulte o arquivo principal:

👉 [**README.md principal (na raiz do projeto)**](../README.md)

### Rodando apenas o Frontend (Desenvolvimento local)

Se desejar rodar apenas o frontend fora do Docker:

1.  Instale as dependências: `npm install`
2.  Inicie o servidor de desenvolvimento: `npm run dev`
